# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Alfresco MCP Server, please report it to using the methods described in [How to Report Security Issues for Alfresco](https://connect.hyland.com/t5/alfresco-blog/how-to-report-security-issues-for-alfresco/ba-p/493939)

Please do not report security vulnerabilities through public GitHub issues.

## Response Time

We aim to respond to security reports within 72 hours.
